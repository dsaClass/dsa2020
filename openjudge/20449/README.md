# 20449: 是否被5整除

## 总时间限制: 10000ms  内存限制: 65536KB

### 描述

给定由0 和 1 组成的字符串 A，我们定义 N_i：从 A[0] 到 A[i] 的第 i 个子数组被解释为一个二进制数。返回0和 1 组成的字符串 answer，只有当 N_i 可以被 5 整除时，答案 answer[i] 为 1，否则为 0。

### 输入

一个0和1组成的字符串

### 输出

一行长度等同于输入的0和1组成的字符串

### 样例输入
```
011
```

### 样例输出
```
100
```

### 提示

- 0可以被5整除->1
- 01不可以被5整除->0
- 011不可以被5整除->0
- 结果是100
